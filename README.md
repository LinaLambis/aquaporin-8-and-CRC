# R Code for Statistical Analysis

This repository contains the R script used to perform the statistical analyses reported in the associated research article.

## Study Overview

The analyses evaluate differential gene expression and its association with clinicopathological outcomes in colorectal cancer, using non-parametric testing and logistic regression models stratified by tumor location.

## Reproducibility and Code Availability

All analyses were conducted in **R (version 4.5.0)**.  
The complete analytical workflow is openly available in this repository to support transparency and reproducibility.

## Data Availability

The data used in this analysis are not publicly deposited due to ethical and institutional restrictions but are **available from the corresponding author upon reasonable request**.
